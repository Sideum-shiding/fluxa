Старт

Что такое Fluxa

Простой синтаксис

Отступы, двоеточия, читаемые блоки и минимум шума.

Статические типы

Int, Float, String, Bool и другие типы языка.

Нативный рантайм

Fluxa запускается через собственный fluxa.exe, без Python.

Первый шаг

Первый файл и правило main

В Fluxa исполняемый файл должен содержать функцию main как первое не-import выражение верхнего уровня. Импорты можно размещать в начале.

Верхнеуровневые объявления функций, структур и импортов допустимы, но действия вроде print(...) и let ... нужно переносить внутрь main.

hello.flx 
import math

func main() -> Void:
    print("Hello, Fluxa!")

Синтаксис

Базовые конструкции языка

Переменные

let для неизменяемых, var для изменяемых.

func main() -> Void:
    let name: String = "Mira"
    var age: Int = 20
    age = 21  # ok
    name = "Anna"  # ошибка

Функции

Объявляй функции отдельно и вызывай их из main.

func main() -> Void:
    print(add(2, 3))

func add(a: Int, b: Int) -> Int:
    return a + b

Условия

Условия строятся так же, как в Python: через двоеточие и отступы.

func main() -> Void:
    let score: Int = 91

    if score >= 90:
        print("excellent")
    else:
        print("keep going")

Циклы

Поддерживаются for и while.

func main() -> Void:
    for i in range(0, 3):
        print(i)

    let counter: Int = 2
    while counter > 0:
        print(counter)
        counter = counter - 1

Коллекции

List<T> и Map<K, V> с методами.

func main() -> Void:
    let nums: List<Int> = [1, 2, 3]
    nums.append(4)
    print(nums.length)

    let user: Map<String, Int> = {"age": 20}
    print(user["age"])

Строки

Интерполяция и методы строк.

func main() -> Void:
    let name: String = "Mira"
    print("Hello {name}")
    print(name.length)
    let num: Int = "42".to_int()

Обработка ошибок

try/catch для исключений.

func main() -> Void:
    try:
        throw "Error!"
    catch e:
        print(e)

Структуры

Для моделей данных используй struct с методами.

func main() -> Void:
    let user = User("Mira", 28)
    print(user.greet())

struct User:
    name: String
    age: Int = 0

    func greet(self) -> String:
        return "Hi, " + self.name

Optional типы

Nullable типы с ?.

func main() -> Void:
    let name: String? = nil
    if name != nil:
        print(name)

Библиотека

Библиотека интерфейсов Fluxa UI

Библиотека fluxaui предоставляет функции для создания текстовых интерфейсов: заголовки, разделы, поля, кнопки и сообщения.

Для ввода от пользователя используйте функцию input или prompt из библиотеки. Сначала объявите переменную, затем вызовите ввод.

fluxaui.flx 
import strings

func line(width: Int) -> String:
    import strings
    return strings.repeat("=", width)

func title(text: String) -> Void:
    print("")
    print(line(52))
    print("  " + text)
    print(line(52))

func prompt(label: String) -> String:
    return input(label + ": ")

func main() -> Void:
    let x: Int = 0
    let input_str: String = prompt("Enter a number")
    x = input_str.to_int()
    print("[ok] You entered: " + x.to_string())

Объявление переменных

Переменные объявляются через let с указанием типа.

let x: Int

Ввод от пользователя

Используйте input для чтения строки.

let name: String = input("Enter your name: ")

Меню выбора

Функция menu для интерактивных меню.

let choice = menu("Main Menu", ["Option 1", "Option 2"])
print("Selected: " + choice.to_string())

Примечание: Требует реализации списков в runtime

Формы

Сбор данных через form.

let data = form(["Name", "Age"])
print(data["Name"])

Примечание: Требует реализации списков в runtime

Прогресс-бар

Отображение прогресса.

progress_bar(50, 100, 20)

Таблицы

Вывод табличных данных.

table(["Name", "Age"], [["Alice", "25"], ["Bob", "30"]])

Примечание: Требует реализации списков в runtime

Да/нет вопросы.

Полный пример приложения

Пример консольного приложения с использованием UI библиотеки:

import fluxaui

func main() -> Void:
    title("Добро пожаловать в демонстрационное приложение!")
    
    section("Основные функции")
    text("Это демонстрация возможностей UI библиотеки Fluxa")
    
    let name = prompt("Введите ваше имя")
    field("Имя пользователя", name)
    
    success("Привет, " + name + "!")
    
    progress_bar(0, 100, 10)
    text("Обработка данных...")
    progress_bar(50, 100, 10)
    text("Сохранение...")
    progress_bar(100, 100, 10)
    
    if confirm("Показать дополнительную информацию?"):
        card("Информация", "Это пример использования UI компонентов")
        success("Информация показана")
    else:
        warning("Информация скрыта")
    
    alert("Демонстрация завершена")
    success("Спасибо за использование приложения!")

Библиотека

Стандартная библиотека Fluxa

Математика

Модуль math для вычислений.

import math

func main() -> Void:
    print(math.sqrt(9))
    print(math.sin(math.pi))

Случайные числа

Модуль random.

import random

func main() -> Void:
    print(random.int(1, 10))
    print(random.choice([1, 2, 3]))

Строки

Расширенные методы строк.

func main() -> Void:
    print("hello".upper())
    print("a,b,c".split(","))

Коллекции

List, Map, Set.

func main() -> Void:
    let s: Set<Int> = {1, 2, 3}
    s.add(4)
    print(s.contains(2))

Файлы

Работа с файлами через io.

import io

func main() -> Void:
    io.write("test.txt", "Hello")
    print(io.read("test.txt"))

JSON

Парсинг и сериализация.

import json

func main() -> Void:
    let data = json.parse('{"a": 1}')
    print(json.stringify(data))

Массивы

Векторные операции как в numpy.

import arrays

func main() -> Void:
    let a = arrays.array([1, 2, 3])
    let b = a * 2
    print(b.sum())

Функциональное программирование

Методы на коллекциях.

func main() -> Void:
    let nums = [1, 2, 3]
    let doubled = nums.map(x -> x * 2)
    print(doubled)

Async

Асинхронный код через async и await

Если точка входа асинхронная, можно сделать её async func main() -> Void:. Fluxa вызовет её сам.

Модуль asyncs даёт простую функцию sleep для демонстрации пауз и ожидания.

async_demo.flx 
async func main() -> Void:
    let message: String = await fetch_message()
    print(message)

async func fetch_message() -> String:
    import asyncs
    print("Loading...")
    await asyncs.sleep(1.0)
    return "Data received"

Стандартные модули

    print("square =", square(value))

Пакеты

Менеджер пакетов Fluxa

Fluxa поддерживает установку пакетов через менеджер пакетов. Пакеты хранятся в fluxa_modules/.

Зависимости указываются в fluxa.json.

fluxa.json 
{
  "name": "myapp",
  "dependencies": {
    "web": "1.0.0",
    "utils": "2.1.0"
  }
}

CLI

Как запускать и проверять Fluxa

Запуск 
fluxa run app.flx
Проверка 
fluxa check app.flx
Сборка 
fluxa build app.flx
Установка пакета 
fluxa install web
Удаление пакета 
fluxa remove web
Версия 
fluxa version

Команда check удобна для редактора и обучения: она проверяет синтаксис и правило main без создания лишних файлов.

Пример

Полный пример программы

full_example.flx 
import json
import arrays
import io
import asyncs

async func process_data(data: List<Any>) -> List<Float>:
    let nums = arrays.array(data)
    let processed = nums * 2 + 1
    await asyncs.sleep(0.1)
    return processed.to_list()

func main() -> Void:
    try:
        let json_text = io.read("data.json")
        let data: Map<String, Any> = json.parse(json_text)
        let values: List<Any> = data["values"]
        
        let result = await process_data(values)
        
        let output = {"processed": result}
        io.write("result.json", json.stringify(output))
        
        print("Data processed and saved!")
    catch e:
        print("Error: " + e.message)
